# Url Shortner Nuxt.js

> A pretty straight forward and easy setup.

## Build Setup

``` bash
# install dependencies
$ yarn install

# copy .env.example to .env
$ cp .env.example .env

# edit .env file with api url
$ nano .env

API_URI=http://shortner.test/api

# serve with hot reload at localhost:3000
$ yarn dev
```
